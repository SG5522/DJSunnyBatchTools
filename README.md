### BatchIDnumber  
批次處理基本資料、證件、人像的Idnumber重複問題。

由於陽信資料庫的人像與證件均使用Idnumber當Key，

故此籌備處與聯名戶的資料在轉換成公司戶或是其他的資料時會連同<font color=#FF0000>個人戶</font>一併更新

需要將現行的其他戶(99)、聯名戶(5)、籌備處(6) 加入~編號處理。 Ps. (號碼) 是他的客戶類別號碼

````
例：
	重複一筆 A123456789~1
	重複二筆 A123456789~2 以此類推
````
